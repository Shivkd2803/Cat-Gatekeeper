Cat Gatekeeper - Work Timer with Cat Break Enforcer
====================================================

HOW IT WORKS:
1. Set your work time (e.g. 25 minutes)
2. Set your break time (e.g. 5 minutes)
3. Click "Start" and focus on your work
4. When work time ends, a REAL CAT PHOTO takes over your FULL SCREEN
5. You cannot dismiss it until the break timer runs out
6. Click "Skip break" only if absolutely necessary

FIRST TIME SETUP (one-time only):
----------------------------------
1. Make sure Python is installed: https://www.python.org/downloads/
   (check "Add Python to PATH" during install)
2. Double-click: install_and_run.bat

AFTER FIRST SETUP:
------------------
Double-click: run.bat

REQUIREMENTS:
- Windows 10 or 11
- Python 3.8+ (free from python.org)
- Internet connection (to load cat photos during breaks)

Enjoy your breaks! The cat demands it.
